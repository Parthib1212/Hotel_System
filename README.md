#  Java Hotel Management System
A lightweight, terminal-based management system designed for hotel administrators to manage rooms, guests, and bookings in real-time. This project demonstrates the transition from in-memory storage to a persistent SQLite database using Java JDBC.

##  Features
* **Persistent Storage**: Uses SQLite to ensure your data survives even after the application is closed.
* **Room Management**: Add new rooms, track occupancy status, and list available rooms.
* **Booking System**: Seamlessly link guests to rooms with automatic price calculation.
* **Advanced Search**: Locate bookings by guest name (partial matching) or find specific rooms by their number.
* **Detailed Reporting**: View a full list of active bookings and financial summaries.

## Technology Stack
* **Language**: Java (JDK 17+)
* **Database**: SQLite
* **Library**: SQLite JDBC Driver
* **IDE**: IntelliJ IDEA
